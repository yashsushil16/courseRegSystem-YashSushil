HOW TO START THE PROJECT
========================

1. Install dependencies:
   cd backend
   npm install
   cd ../frontend
   npm install

2. Start backend:
   cd backend
   node server.js

3. Start frontend (new terminal):
   cd frontend
   npm start

4. Open browser:
   http://localhost:3000

5. Login:
   Student: student1@yituniversity.edu / student123
   Faculty: faculty1@yituniversity.edu / faculty123

6. Seed data (optional):
   cd backend
   npm run seed
