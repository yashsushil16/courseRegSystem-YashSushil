## Screenshot Tips

- Use PNG format for best quality
- Recommended size: 1920x1080 or 1280x720
- Capture full browser window or specific sections
- Ensure UI looks clean and professional
- Use consistent browser/theme for all screenshots

## How to Take Screenshots

### Windows
- Press `Win + Shift + S` for Snipping Tool
- Or use `Alt + Print Screen` for active window

### Mac
- Press `Cmd + Shift + 4` for selection
- Press `Cmd + Shift + 3` for full screen

### Browser Extensions
- Use browser extensions like "Awesome Screenshot" or "Nimbus Screenshot"
